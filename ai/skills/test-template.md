# Test Template (Mandatory Output)

For any proposed new test, output must include:

- File path (where to add)
- Arrange:
  - entry screen
  - data/state assumptions
- Act:
  - remote steps
- Assert:
  - focus checkpoints
  - outcome checkpoints (screen, overlay, player, etc.)
- Flake risks + mitigation
- Required new methods (flow/page/component) (allowed folders only)
